System Architecture: The Hybrid FEC-ARQ Engine

This implementation handles extreme jitter and packet loss using a low-overhead hybrid technique:

  • XOR Parity (FEC): Redundancy overhead of exactly 50% (1.5x total baseline bandwidth) covers the vast majority of single-packet drops instantly with zero additional latency.
  • NACK Engine: If both the primary media frame and the parity block are lost in transit, the receiver requests targeted retransmissions. These requests are safely bounded within the 80ms playout delay window.
